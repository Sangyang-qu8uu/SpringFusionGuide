# spring-mq

## 项目模块说明：

- spring-amqp
  demo包下都是原生使用，常用5种队列模式，Rpc用的比较少就没有再列出。
  1.simplest  （简单模式）
  2.workqueue（工作队列）
  3.publishsubscribe(发布订阅模式)
  4.direct（路由模式）
  5.topic(通配符模式)
  6.延迟队列（特殊模式）
  amqp包下采用注解开发

- spring-integration

  mqtt客户端以及服务端
- spring-kafka

  spring简单整合kafka
